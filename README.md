## Commands

| Command                     | Description               |
| --------------------------- | ------------------------- |
| `TokeBloc: New Toke Bloc`   | Generate a new Toke Bloc  |

## Code Actions

| Action                               | Description                                                            |
| ------------------------------------ | ---------------------------------------------------------------------- |
| `Wrap with TokeBlocBuilder`          | Wraps current widget in a `TokeBlocBuilder`                            |
| `Wrap with TokeBlocSelector`         | Wraps current widget in a `TokeBlocSelector`                           |
